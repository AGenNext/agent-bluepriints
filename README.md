# Agent Blueprints

Build production-ready AI agents for OpenMetadata.

Agent Blueprints is a starter kit for creating chat-based data agents that understand metadata, lineage, governance workflows, and catalog operations.

## What is included

- Agent specifications in `agents/`
- TypeScript SDK and chat UI in `sdk/`
- Marketing website in `site/`
- Repository hardening files in `.github/`

## Development

Install dependencies and run the workspace scripts from the repository root.

```bash
npm install
npm run build
```

Run the marketing site locally:

```bash
cd site
npm install
npm run dev
```
